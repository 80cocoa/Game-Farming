# Game-Farming
A 2d farming simulation game built with Unity
内容包含：
- 背包系统
- 种植生长系统
- NPC 自定义 Schedule 生活系统
- 动画系统
- 
技术栈：
- ObjectPool API
- AStar 寻路算法
- MVC 设计模式
- 观察者模式
- Particle System 粒子效果
- UI Toolkit
- 物理检测 API
- Animator Animation
